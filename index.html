Este componente es una tarjeta básica con una imagen en la parte superior, un título, un texto y un botón transparente que al posicionar el ratón adquiere un color y sombra. La imagen también tiene una sombra y un efecto al pasar el ratón que la trae hacia el frente. Además este tablón con pestañas es otro componente, es un efecto de cristal opaco con la información de los estilos utilizados

Para traer las fotos al frente usamos transform: scale(1.0) y en función de los parametros x e y aumenta más o menos de tamaño.

Para conseguir una sombra interior debemos añadir inset a box-shadow asi conseguimos un efecto mas cercano al cristal opaco. Para poder conseguir el tablón con esta textura debemos tener un div y añadira la clase la propiedad after, ej: .tablero-cristal::after y aplicar un desenfoque(filter: blur(10px)). Es importante heredar el background para que el efecto funcione.

Dependiendo del la posición que le demos a las pestañas (ej: relative) se pueden superponer al tablero usando un z-index cuando la pestaña esté activa, las demás estarán por detrás y se pueden posicionar moviendolas por ejemplo con bottom, esto combinado con un border-bottom (ej: elegir el color de fondo del tablero del mismo color) nos permite un efecto que al pinchar en la pestaña se combinen pestaña y tablero, no se ve el borde inferior pero si los laterales.
Para el tablero: determinar la distancia entre los margenes del elemento y de su bloque contenedor usamos las propiedades left, rigth, bottom y top, así cada pestaña mantiene su posicion y oculta a las demás, si queremos un efecto en el que se vean una encima de otra y se vea un poco el borde podemos quitar estas propiedades o darles valores diferentes

Estás autorizado para ctrl + c y ctrl + v ; D

/*Propiedades y tamaño del tablero de cristal*/
.tablero-cristal {
  margin: auto;
  position: relative;
  min-width: 500px;
  max-width: 100vh;
  height: 350px;
  background: inherit;
  border-radius: 5px;
  overflow: hidden; /*ocultamos el efecto de desenfoque si se sale de la caja */
  box-shadow: inset 20px 20px 200px -70px #B3B6B7; /*sombra interior, asi logramos un efecto mas cercano a un cristal*/
}

/*Efecto de cristal opaco*/
.tablero-cristal:after {
  content: '';
  width: 100%;
  height: 100%;
  background: inherit; 
  position: absolute;
  top: 0 ;
  filter: blur(10px);
}

/*Sobreponemos el contenido por delante del desenfoque*/
.contenido-cristal {
  padding: 30px;
  position: relative;
  z-index: 1;
}

/*Propiedades y tamaño del tablero de pestañas*/
.tabs {
  position: relative;   
  min-height: 200px;
  margin: 25px ;
}

/*Estilos pestañas*/
.tab {
  float: left; /*flotamos las pestañas para tenerlas juntas en linea*/
}

.tab label {
  color: white;
  padding: 10px; 
  border-radius: 10px;   
  position: relative;
  bottom: 10px; /*alejamos las pestañas de los contenidos*/
  box-shadow: inset 2px 2px 15px -7px #B3B6B7
} 


.tab [type=radio] {
  display: none;   /*ocultamos el input radio*/
}

[type=radio]:checked ~ label {
  background: #ddd; /*color pestaña activa*/
  z-index: 2; /*aumentamos el index para verla por encima del tablero de contenidos*/
}

/*Propiedades y tamaño del tablero de contenidos*/
.content-tab {
  color: white;
  /*Para determinar la distancia entre los margenes del elemento y de su bloque contenedor. usamos las propiedades left, rigth, bottom y top, asi cada pestaña mantiene su posicion y oculta a las demás, si queremos un efecto en el que se vean una encima de otra y se vea un poco el borde podemos quitar estas propiedades o darles valores diferentes */
  position: absolute; /*Posicion para que ocupe siempre el mismo sitio al seleccionar la pestaña*/
  top: 30px; /*lo alejamos de la parte superior*/
  left: 0;
  right: 0;
  bottom: 0;
  padding: 20px;
}


/*Efecto al mostrar el contenido*/
.content-tab > * { 
  opacity: 0; /*Asi ocultamos el contenido de las demas pestañas y evitamos que se solapen, ya que no tenemos un background color que las oculte*/

  -webkit-transform: translateX(-100%);
  -moz-transform:    translateX(-100%);
  -ms-transform:     translateX(-100%);
  -o-transform:      translateX(-100%);
  -webkit-transition: all 0.7s ease;
  -moz-transition:    all 0.7s ease;
  -ms-transition:     all 0.7s ease;
  -o-transition:      all 0.7s ease;
}

[type=radio]:checked ~ label ~ .content-tab { 
  z-index: 1;
  overflow: auto; /*Si el texto excede se añade la barra de scroll*/
}

/*Posicion inicial del contenido*/
[type=radio]:checked ~ label ~ .content-tab > * {
  /*Incluimos la posicion inicial para asegurarnos que el efecto de transicion se ejecute correctamente cuando estemos mostrando el contenido de una pestaña*/
  opacity: 1;
  -webkit-transform: translateX(0);
  -moz-transform:    translateX(0);
  -ms-transform:     translateX(0);
  -o-transform:      translateX(0);
}
